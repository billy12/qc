# Claude Code — Available Skills

These skills are available in this Claude Code session via slash commands.
Invoke any of them by typing `/<skill-name>` or asking Claude to use them by name.

---

## Resume & Job Application Skills

| Skill | Command | What It Does |
|-------|---------|--------------|
| Resume ATS Optimizer | `/resume-ats-optimizer` | Optimize resume for Applicant Tracking Systems, check ATS compatibility, analyze keyword match |
| Resume Bullet Writer | `/resume-bullet-writer` | Transform weak resume bullets into achievement-focused statements with metrics and impact |
| Resume Formatter | `/resume-formatter` | Ensure ATS-friendly formatting and create clean scannable layouts |
| Resume Quantifier | `/resume-quantifier` | Find opportunities to add metrics and estimate numbers when exact data unavailable |
| Resume Section Builder | `/resume-section-builder` | Create targeted resume sections optimized for different experience levels and roles |
| Resume Tailor | `/resume-tailor` | Customize resume for specific job postings while maintaining truthfulness |
| Resume Version Manager | `/resume-version-manager` | Track different resume versions, maintain master resume, manage tailored versions |
| Executive Resume Writer | `/executive-resume-writer` | Create C-suite and VP level resumes emphasizing strategic leadership |
| Tech Resume Optimizer | `/tech-resume-optimizer` | Optimize resumes for software engineering, PM, and technical roles |
| Creative Portfolio Resume | `/creative-portfolio-resume` | Balance visual design with ATS compatibility for creative roles |
| Academic CV Builder | `/academic-cv-builder` | Format CVs for academic positions with publications, grants, and teaching |

## How to Use in a New Chat

Start a new Claude Code session, share `PROFILE.md` and `HANDOVER.md`, then invoke any skill:

```
/interview-prep-generator
```

Or just describe what you need:
> "Use the interview prep generator with my profile to build STAR stories for the Queen Creek rec coordinator role"
